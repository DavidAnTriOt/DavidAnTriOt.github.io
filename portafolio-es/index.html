HABILIDADES TECNOLÓGICAS

+

HTML 5

Sus siglas vienen del inglés 'HyperText Markup Language'. Traducido al español quiere decir 'Lenguaje de Marcado e HiperTexto'. Este lenguaje de marcado nos permite crear la estructura (o cuerpo) básica de un sitio web mediante tags (etiquetas). El número cinco (5) hace referencia a la versión en la que se encuentra actualmente. Todos los títulos, párrafos, formularios, videos, entre muchísimas cosas más, las tenemos gracias a este maravilloso lenguaje.



Intermedio

+

CSS 3

CSS es un lenguaje de estilo. Sus siglas también tienen un significado 'Cascading StyleSheet', su traducción es 'Hojas de estilo de cascada'. El número tres (3) hace referencia a la versión actual de dicho lenguaje. También nos permite darle estilos al cuerpo de nuestros sitios web. Es considerado como el maquillaje de HTML. Todo el diseño de un sitio va directamente relacionado con este lenguaje.



Intermedio

+

JavaScript

JavaScript es un lenguaje de programación que nos permite añadir, eliminar, aplicar animaciones o efectos a los componentes de nuestro sitio. Es la herramienta perfecta para realizar sitios web dinámicos y muy atractivos de la mano de CSS y HTML. Con el paso del tiempo ha evolucionado de manera exponencial y ahora mismo es absolutamente necesario conocerlo si quieres ser desarrollador web.



Básico

+

Bootstrap

Es una de las estructuras de trabajo más usados en el mundo. Este framework hace mucho más sencillo el diseño y posicionamiento de los elementos con CSS. Te ahorras mucho tiempo gracias a que por defecto él trae las herramientas o código interno para que tú no tengas que escribir nada más que clases específicas que tiene predefinidas. Con una de estas clases de bootstrap te podrías ahorrar cinco o más lineas de código en CSS.



Intermedio

+

Sass

Al igual que CSS, es un lenguaje de estilos que con el tiempo se ha ido extendiendo. Sin embargo no es la "competencia de CSS" sino todo lo contrario. El lenguaje de estilos Sass debe traducirse luego a CSS. Sass nos permite escribir código CSS mucho más fácil, rápida y además facilita herramientas que CSS por si solo no puede darte a pesar de usar los mismos atributos.



Básico

+

WordPress

Wordpress es un sistema de gestión de contenido. En palabras menos técnicas se considera una herramienta que facilita la administración de muchos aspectos de tu sitio web sin necesidad tener conocimientos sobre programación. Cualquier persona puede usar o modificar el software de WordPress, además puede hacerlo de forma gratuita. Más del 32 % de todos los sitios web en internet están hechos con WordPress.



Intermedio

+

Python

Python es un lenguaje de programación de alto nivel, interpretado y versátil. Destaca por su sintaxis clara y legible, lo que lo hace ideal para principiantes y profesionales. Es multiparadigma, permitiendo programación procedural, orientada a objetos y funcional. Su amplia comunidad y biblioteca estándar ofrecen herramientas poderosas para una variedad de aplicaciones, desde desarrollo web hasta inteligencia artificial. Python se utiliza en campos como ciencia de datos, desarrollo de juegos y automatización de tareas.

Intermedio

+

AWS

Amazon Web Services (AWS) es una plataforma de servicios en la nube líder en el mercado, ofreciendo una amplia gama de servicios de cómputo, almacenamiento, bases de datos, análisis, inteligencia artificial, aprendizaje automático, Internet de las cosas (IoT), seguridad y más. Con una infraestructura global y escalable, AWS permite a empresas y desarrolladores implementar aplicaciones de forma rápida y eficiente, pagando solo por los recursos que utilizan.


Básico

+

Android Studio

Android Studio es un entorno de desarrollo integrado (IDE) oficial para crear aplicaciones Android. Basado en IntelliJ IDEA, ofrece herramientas poderosas para desarrolladores, como emuladores de dispositivos, depuración avanzada y soporte para varios lenguajes de programación, incluido Kotlin. Con una interfaz intuitiva y una variedad de plantillas y componentes, Android Studio simplifica el proceso de creación de aplicaciones móviles para dispositivos Android.


Intermedio

+

GitHub

GitHub es una plataforma de desarrollo colaborativo que permite a los equipos de programadores trabajar juntos en proyectos de software. Ofrece control de versiones utilizando Git, seguimiento de problemas, administración de tareas y herramientas de integración continua. Con características como ramificaciones, solicitudes de extracción y comentarios en línea, GitHub facilita la colaboración y la revisión de código entre desarrolladores de todo el mundo.
